# AnyLaunch 随启

> 便携软件启动器 · 双击 Ctrl 唤出 · 即用即走

---

## 一、为什么会有这个工具？

我平时玩 RAMOS，系统每次重启都是全新的，所以软件都放在便携目录里。时间一长，一堆绿色软件、单文件散落在各个文件夹，每次找起来很烦。

于是就写了个启动器凑合用，最近整理了一下觉得还行，分享出来。

如果你也有类似习惯，或许用得上：

- 系统是 **RAMOS**，软件都放便携目录
- 硬盘里一堆 **绿色版、单文件**，懒得每次翻文件夹
- 喜欢 **极简、干净** 的启动方式

---

## 二、它能做什么？

- **双击 Ctrl** 弹出面板，再双击隐藏，随用随走
- **打字即搜**：唤出面板后直接输入关键词，支持中英文、拼音首字母、空格分词
- **鼠标操作**：左键启动软件，右键直接打开文件所在位置（整理快捷方式特别方便）
- **键盘全支持**：方向键选择，回车启动，Esc 关闭
- **自动扫描**：监控指定文件夹下的 `.lnk`、`.exe`、`.bat`，有增删自动刷新
- **自定义图标**：图标太小或模糊？在快捷方式旁边放一个 `.ico` 文件，自动加载为高清图标，强迫症友好

## 三、功能详情

### 全局热键唤出
任意界面下，**连续按两次 Ctrl 键**，即可唤出/隐藏启动面板，无需鼠标点击。

### 图标网格展示
以固定行列的网格形式展示所有应用，图标大小和间距自动适配屏幕分辨率。每个图标下方显示程序名称，超长文字自动裁剪（保留开头）。

### 实时搜索过滤
唤出面板后直接打字，即时过滤应用列表。支持中英文和空格分割的多关键词搜索。按 **Esc** 键清空搜索框或关闭面板。

### 多方式启动
- **单击图标**：选中并高亮
- **双击图标**：直接启动应用
- **Shift + 单击/回车**：启动后保持面板不关闭（方便连续启动多个应用）

### 右键快捷操作
在任意图标上右键单击，自动打开该程序所在文件夹并高亮选中文件（支持快捷方式追踪到真实目标路径）。

### 自动扫描与图标加载
首次启动时自动扫描指定目录（及子目录）中的所有 `.lnk`、`.exe`、`.bat` 文件，自动提取图标。支持自定义 `.ico` 图标文件。

### 其他特性
- 半透明毛玻璃界面，圆角设计
- 多屏记忆，记住上次所在显示器
- 单实例运行，重复启动自动唤出已有面板
- 自动开机启动（可选）

## 五、当前说明

- 监控目录**硬编码**为 `D:\快捷方式`。使用前请将快捷方式集中放在此目录下。
- 如果某个软件图标显示很小，是因为程序自带的图标尺寸有限。在对应程序旁边放一个 `.ico` 文件（大于64x64px），程序会自动加载，图标就会变清晰。
