# AnyLaunch
# 随启 - 便携软件启动器

## 一、软件简介

**随启** 是一款轻量、高效的便携软件启动工具。通过全局热键（**双击 Ctrl**）即可快速唤出一个半透明的图标网格面板，让你从杂乱桌面或文件夹中解脱出来，指尖轻点即可启动常用软件。

适合放置在电脑硬盘 U 盘、移动硬盘或同步盘中，实现“一次配置，随处使用”。

---

## 二、核心功能

### 1. 全局热键唤出
- 任意界面下，**连续按两次 Ctrl 键**，即可唤出/隐藏启动面板。
- 无需鼠标点击任务栏或桌面，专注当前工作流。

### 2. 图标网格展示
- 以固定行列的网格形式展示所有应用。
- 图标大小和间距会自动适配屏幕分辨率，视觉统一。
- 每个图标下方显示程序名称，超长文字自动裁剪（保留开头，如“落雪音乐助手”→显示“落雪音乐”）。

### 3. 实时搜索过滤
- 唤出面板后**直接输入关键词**，即可即时过滤应用列表。
- 支持中英文和空格分割的多关键词搜索（如输入“音乐 助手”可匹配“落雪音乐助手”）。
- 按 **Esc** 键清空搜索框或关闭面板。

### 4. 键盘导航与启动
- **方向键 ↑↓←→** 在图标间快速移动。
- **回车键** 启动当前选中的应用。
- **退格键** 删除搜索文字。

### 5. 多方式启动
- **单击图标**：选中并高亮。
- **双击图标**：直接启动应用。
- **按住 Shift + 单击/回车**：启动后保持面板不关闭（方便连续启动多个应用）。

### 6. 右键快捷操作
- 在任意图标上**右键单击**，自动打开该程序所在文件夹并高亮选中文件（支持快捷方式追踪到真实目标路径）。

### 7. 自动扫描与图标加载
- 首次启动时自动扫描指定目录（及子目录）中的所有 `.lnk`、`.exe`、`.bat` 文件。
- 自动提取程序图标并显示在网格中。
- 支持自定义 `.ico` 图标文件（放入程序目录或快捷方式目录即可自动加载）。

### 8. 目录实时监控
- 当监控目录下新增或删除应用时，面板会自动刷新，无需手动重启。

### 9. 半透明毛玻璃界面
- 背景采用渐变半透明设计，视觉上轻盈且不遮挡视线。
- 窗口圆角处理，风格现代。

### 10. 多屏记忆
- 自动记住面板最后一次所在的显示器，下次启动仍在同一屏幕居中显示。

### 11. 单实例运行
- 再次启动程序时，自动唤出已有面板，不会重复运行多个实例。

### 12. 自动开机启动（可选）
- 程序首次运行会自动添加到开机启动项，确保随时可用。

## 三、使用场景

- **办公场景**：将常用工作软件（Office、浏览器、设计工具等）集中管理，告别桌面混乱。
- **游戏场景**：快速启动游戏平台（Steam、Epic）或加速器。
- **便携设备**：将程序放在 U 盘中，插入任何电脑即可使用自己的软件收藏夹。
- **开发环境**：快速启动 IDE、终端、数据库工具等。

## 五、快速上手

- **唤出/隐藏**：双击 Ctrl
- **搜索**：唤出面板后直接打字
- **导航**：方向键
- **启动**：回车 或 双击图标
- **关闭**：Esc 或 点击面板外区域
- **查看文件位置**：右键点击图标



**随启 —— 让常用软件，触手可及。**
